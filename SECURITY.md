# Security Policy

## Supported releases

Security fixes are considered for the latest public releases:

- Excelsis Helper `1.3.1-public.1`
- ExcelsisView `0.6.6`

Older releases may not receive security updates.

## Reporting a vulnerability

Use this repository's **Security** tab and submit a report through GitHub
Private Vulnerability Reporting. Please do not open a public issue for a
suspected vulnerability.

Include the affected component and version, the expected impact, and concise
reproduction steps. Attach only samples that you created or are permitted to
redistribute. Do not include credentials, customer data, private CAD files, or
private network paths.

Reports are reviewed on a best-effort basis. Please allow time to investigate
and prepare a coordinated fix before publishing details.

## Scope

This policy covers the source and release artifacts published in this
repository. Report vulnerabilities in SOLIDWORKS or another third-party
product to that product's vendor. Test locally and do not disrupt third-party
systems or services.
